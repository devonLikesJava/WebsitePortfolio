The purpose of this program is to take in a temperature from the user and convert it to the 2 other types ranging from Celcius, Fahrenheit, and Kelvin.

The way this program works is it uses the scanner class to get the temperature from the user as well as the unit, then uses else if statements and converts them based on the conditions of the character entered for the units. Also instead of hard coding the math in the statements I used methods for each of the conversions.
