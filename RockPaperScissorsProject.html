The purpose of this program is to create a text based rock, paper, scissors game against a computer.

The way this program works is it uses the scanner class to get the input from user, based on input that decides the chosen move using case statements. Then the computer gets a randomly generated number from 1-3 and that decides it's move using case statements. Then both moves are compared using if else statements and depending on whether they are the same it will output, tie, win, computer wins.
