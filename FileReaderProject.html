The purpose of this program is to return the number of lines, words, and characters from a txt file.

The way this program works is it uses the input method to get file from user then uses open and read method to access the file. Stores the readfile method output into a variable and that variable is ran through split, and splitlines method to split the words and lines. Those are then encapsulated by a len method that counts the words and lines which is then printed out.
